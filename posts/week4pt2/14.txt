  <div class="vtbegenerated">
   <p>Reviewing text mining I wanted to find out how the process looks like in terms of steps from loading required packages, data, selecting creating objects and etc… Initially I wanted to see basic examples to understand the possible steps.</p> 
   <p>Below articles helped:</p> 
   <p><a href="https://towardsdatascience.com/text-mining-for-dummies-text-classification-with-python-98e47c3a9deb">https://towardsdatascience.com/text-mining-for-dummies-text-classification-with-python-98e47c3a9deb</a></p> 
   <p><a href="https://medium.com/towards-artificial-intelligence/text-mining-in-python-steps-and-examples-78b3f8fd913b">https://medium.com/towards-artificial-intelligence/text-mining-in-python-steps-and-examples-78b3f8fd913b</a></p> 
   <p><a href="https://www.datacamp.com/community/tutorials/text-analytics-beginners-nltk">https://www.datacamp.com/community/tutorials/text-analytics-beginners-nltk</a></p> 
   <p>I am interested in social media data, there are many different articles and tutorials out there.</p> 
   <p><a href="https://www.toptal.com/python/twitter-data-mining-using-python">https://www.toptal.com/python/twitter-data-mining-using-python</a></p> 
   <p>I like this article because it shows how to access the twitter data including tweets from a specific account. I am not sure about some of the code as grabbing the tweets with for loop and printing sounds costly to me. I like the fact that there is an example of NYT tweets but I don’t like that there is no analysis of the text , it is simply mining the data.</p> 
   <p><a href="https://towardsdatascience.com/creating-the-twitter-sentiment-analysis-program-in-python-with-naive-bayes-classification-672e5589a7ed">https://towardsdatascience.com/creating-the-twitter-sentiment-analysis-program-in-python-with-naive-bayes-classification-672e5589a7ed</a></p> 
   <p>This article captures on access to twitter data along with creating the corpus file and tokenizing and analysis using nltk package. I also like the fact that the provided approach gives us a classification model using Naïve Bayes.</p>
